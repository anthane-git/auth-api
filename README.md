# Customer identity access management service.

## Running the app

```bash
    # development
    $ npm run start

    # watch mode
    $ npm run start:dev

    # production mode
    $ npm run start:prod
```

## Running database migrations (development only)

```bash
npx prisma migrate dev --name add ADD_COMMENT_HERE
npm run db:migrate
```

## Database Admin Tool (prisma studio)

```
npm run db:show
```
